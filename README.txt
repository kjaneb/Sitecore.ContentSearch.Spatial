Fork of the original Sitecore.ContentSearch.Spatial by Ahmed Okour (https://github.com/aokour/Sitecore.ContentSearch.Spatial) with the following changes:

* Works with Sitecore 8.1.x
* Addition of WithinBounds(double minLatitude, double minLogitude, double maxLatitude, double maxLogitude) method
* Support for a "SwitchOnRebuildLuceneIndexWithSpatial" Index Type
* Switched to kms instead of miles
